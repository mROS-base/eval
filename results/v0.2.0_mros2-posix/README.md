# v0.2.0_mros2-posix

## Experiment Conditions

- Both pub/sub node and echo node were run on Ubuntu 20.04.4 on Raspberry Pi 4 Model B (4GB RAM)
- Machines were connected via Ethernet cables and a router (WSR-5400AX6S/DMB, Buffalo Inc.)

## Directories names
